# AHNotify

WoW Classic TBC (2.5.5) addon that monitors your Auction House listings.

## Features

- Scans your auctions when you open the Auction House
- Auto-refreshes every 10 seconds while AH is open
- Groups identical items (same name + same buyout price)
- Notifies in chat when auctions are sold
- **Restock alerts** when all auctions in a group are sold
- Minimap icon to open the auction list window
- Scrollable GUI window with item icons, quantities, prices, time left, and status

## Commands

- `/ahnotify` or `/ahn` - Toggle the auction list window
- Left-click minimap icon - Toggle auction list
- Right-click minimap icon - Lock/unlock icon position

## Installation

Copy the `AHNotify` folder to `Interface/AddOns/` in your WoW Classic directory.
